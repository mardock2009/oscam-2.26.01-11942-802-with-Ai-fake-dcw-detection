## ECM Length List

Starting with SVN [r5133](https://trac.streamboard.tv/oscam/changeset/5133 "Add new parameter ecmwhitelist for physical readers and remove ...") you can add a filter on **hardware** readers so only valid ECM´s are sent to your reader. Later, the possibility to add network readers (proxies) was added. However, please note that the values beneath should only be used in your configuration if necessary (failing ecms). Don't try to define filters for all providers in the world for your network readers.

Where/how this parameter is used you can find here: [​https://wiki.streamboard.tv/wiki/OSCam/en/Config/oscam.server#ecmwhitelist](https://wiki.streamboard.tv/wiki/OSCam/en/Config/oscam.server#ecmwhitelist)

Please set only the working **Provider ID** and do **not** set any RSA Card

Please update also [​http://streamboard.de.vu:8001/wiki/OscamWhiteList](//http://streamboard.de.vu:8001/wiki/OscamWhiteList) when change something here.
