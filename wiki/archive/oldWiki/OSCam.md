# Welcome to OSCam Wiki

OSCam is an **O**pen **S**ource **C**onditional **A**ccess **M**odule software, based on the very good MpCS version 0.9d created by [​dukat](https:https://board.streamboard.tv/user/33325-dukat/).

# Getting and building OSCam

+   See [GettingOscam](https://trac.streamboard.tv/oscam/wiki/GettingOscam) page for download information.
+   See [BuildingOscam](https://trac.streamboard.tv/oscam/wiki/BuildingOscam) page for information on how to build OSCam.
+   See [CrossCompiling](https://trac.streamboard.tv/oscam/wiki/crosscompiling) page for more info on cross compilation and toolchains.

# Documentation: Parameters and examples

The description of all valid parameters can be found in [​distribution docs](https:https://svn.streamboard.tv/oscam/trunk/Distribution/doc/txt/) and on the [​Wiki](https:https://wiki.streamboard.tv/wiki/OSCam)
Configuration examples are located in [​distribution examples](https:https://svn.streamboard.tv/oscam/trunk/Distribution/doc/example/) and [in the forum](http://www.streamboard.tv/wbb2/thread.php?threadid=26659)
Informations about the [WebIf](https://trac.streamboard.tv/oscam/wiki/WebIf) and XMLApi can be found [​here](https:https://trac.streamboard.tv/oscam/wiki/WebIf)

# Documentation: Card list and card reader

Here you can find the schema for a [CardReader](https://trac.streamboard.tv/oscam/wiki/CardReader) and a list of the supported [CardReader](https://trac.streamboard.tv/oscam/wiki/CardReader).
We will try to keep a list of supported cards in [CardsList](https://trac.streamboard.tv/oscam/wiki/CardsList).
We will try to keep a list of supported Devices/ STB's in [DeviceList](https://trac.streamboard.tv/oscam/wiki/DeviceList).
We will try to keep a list of valid ECM lengths in [ECMLengthList](https://trac.streamboard.tv/oscam/wiki/ECMLengthList).
We will try to keep a list of valid ECM headers in [HeaderWhiteList](https://trac.streamboard.tv/oscam/wiki/HeaderWhiteList).
